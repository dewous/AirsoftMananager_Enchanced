# Страйкбол Менеджер

Веб-приложение для управления составом страйкбольной команды и просмотра тематических новостей.

## Стек

- React 19 + Vite 8
- React Router DOM 7
- Wikipedia API (новости)
- LocalStorage (сохранение данных)

## Требования

- Node.js 18 или новее
- npm 9 или новее

## Установка и запуск

npm install

npm run dev
